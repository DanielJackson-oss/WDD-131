Overview

Purpose

BudgetJar is a simple, browser-based budget tracker that lets a user set a spending limit, log individual expenses against it, and immediately see how much money they have left. The purpose is to replace mental math or scattered notes with one clear, visual snapshot of spending.

Audience

The intended audience is students and young adults managing a limited weekly or monthly amount — like spending money, groceries, or entertainment funds — who want something fast and login-free to check from their phone.

Dynamic elements

JavaScript drives all of the page's interactivity. Expenses are stored as objects (description, amount, category) inside an array. Submitting the "add expense" form triggers an event listener that creates a new expense object, pushes it into the array, and re-renders the list. The running total is calculated with the array's reduce() method, and the remaining balance updates and changes color (conditional branching) as the user nears or exceeds their budget. A category filter uses filter() to show only matching expenses, and a delete button on each entry removes it from the array and the DOM using forEach() to re-render. A visual progress bar ("budget jar") fills based on percentage of budget spent, giving an at-a-glance sense of where the user stands.

Navigation

BudgetJar is a single-page application, so there is only one page in the navigation.

Home

Content

Home page

The home page contains everything BudgetJar does, since it's a one-page app:

  • Header with app name and tagline
  • Budget-setup control to enter the total budget amount
  • Visual "jar" progress bar showing spent vs. remaining, with an image of a jar/piggy bank
  • Add-expense form (description, amount, category)
  • Dynamically rendered expense list, each entry removable
  • Category filter buttons
  • Summary stats: total spent, remaining balance, percent of budget used
  • Footer with name and last-updated date
